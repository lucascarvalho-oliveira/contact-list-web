# contact-list-web 
Contact List project developed using HTML5 and CSS3.
It displays contacts with photo, name, email, and buttons to edit or delete.
The project focuses on responsive layout, Flexbox, image styling, and custom typography.

## Technologies
- HTML5
- CSS3
- Flexbox
- Google Fonts:
  - Roboto

## How to Run
1. Clone this repository:
git clone <https://github.com/carvalho484/Projeto_4_HTML_CSS-.git>
2. Open the index.html file in any modern web browser.
3. To add new contacts, copy a <li class="list">...</li> inside the <ul class="container_list"> and update the image and information.

## Features
- Contact list including:
  - Circular profile photo
  - Name and email
- Edit and delete buttons for each contact (visual only, no backend functionality)
- Responsive layout using Flexbox
- Soft color palette and rounded borders
- Custom font via Google Fonts
